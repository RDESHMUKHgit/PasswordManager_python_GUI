# PasswordManager_python_GUI
A GUI-based Password Manager built with Python and Tkinter featuring secure password generation, JSON-based credential storage, password search, and automatic clipboard copying.

## Features

- Generate strong random passwords
- Copy passwords automatically
- Save credentials in JSON
- Search saved passwords
- Simple Tkinter GUI

## Technologies

- Python
- Tkinter
- JSON
- Pyperclip
